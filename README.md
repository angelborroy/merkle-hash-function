# merkle-hash-function

Sample algorithm implementing Merkle–Damgård hash function for teaching purposes.

## Requirements

* Java 11
* Maven 3.5

## Building

```
$ mvn clean package
```

## Running

```
$ java -cp target/classes es.usj.crypto.HashApp

Message: 01111010011111110100101111111011
Digest:  11001110
```
